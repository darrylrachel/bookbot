BookBot is a simple Python text analysis tool that reads `.txt` files and prints word and character frequency stats.  
Built as part of the **Boot.dev "Build a Bookbot" Guided Project**.

---

## Features
- Reads text files of any size
- Counts word and character frequency
- Sorts words by frequency
- Outputs a clean report to the console

---

## Example Usage
```bash
python main.py books/frankenstein.txt

--- Begin report of books/frankenstein.txt ---
10292 words found in the document
The most common letter is: e (5151 times)
```
--- End report ---


## Technologies

- Python 3

- File I/O

- Dictionaries & Loops

- Sorting algorithms
